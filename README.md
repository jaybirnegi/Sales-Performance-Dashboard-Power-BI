# 📊 Sales Performance Dashboard (Power BI)

## 🔍 Overview

This project presents an interactive Power BI dashboard designed to analyze sales and profit performance across regions and product categories. It focuses on delivering business insights through KPI tracking, trend analysis, and interactive visualizations.

---

## 🎯 Objectives

* Analyze overall sales and profit performance
* Identify top-performing regions and product categories
* Track trends over time
* Enable data-driven decision-making

---

## 📌 Key Features

* KPI cards for **Total Sales, Profit, and Units Sold**
* Sales breakdown by **category and region**
* Profit analysis across product categories
* Time-series analysis for sales trends
* Interactive filters (Year, Region)

---

## 📊 Insights

* Identified high-performing product categories based on profit
* Analyzed regional sales contribution and distribution
* Observed sales growth trends and fluctuations over time
* Highlighted key business insights for performance evaluation

---

## 🛠 Tools Used

* Power BI
* Data Visualization
* Data Analysis
